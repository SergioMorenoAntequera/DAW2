Ejercicios de ampliación de PHP

1. Crea un programa PHP que genere una página con N cuadrados de colores aleatorios que se coloquen también de forma aleatoria por la pantalla. Cada cuadro mide 50x50 píxeles.

2. Escribe un programa en PHP que lea un texto y determine si es o no un palíndromo.

3. Escribe un programa en PHP para mostrar la lista de números primos entre 1 y N, donde N es un número introducido por el usuario.

4. Escribe un programa que muestre la lista de los primeros N números EMIRPS, siendo N un número introducido por el usuario.

Un EMIRP es un número primo cuyo inverso también es primo. Por ejemplo, 13 es un EMIRP, porque si lo escribes al revés (31) el resultado también es un número primo.

5. Escribe un programa en PHP para jugar al Simon con números

En lugar de colores y sonidos, el juego consistirá en una secuencia de números del 1 al 4.

Inicialmente, el servidor mostrará una secuencia de longitud 1 (por ejemplo "2") durante un instante. Luego la pantalla se borrará y el usuario debe repetir la secuencia.

Luego el servidor mostrará una secuencia de longitud 2 (por ejemplo "24"), y el jugador la intentará repetir. Después se pasará a longitud 3 (por ejemplo, "244"), y así sucesivamente, hasta que el jugador falle.

Observa que la secuencia debe de ser acumulativa. Por ejemplo, "2" -> "24" -> "244" -> "2441" -> "24414", etc. La secuencia será distinta en cada ejecución del juego.

6. Escribe un programa en PHP para emular al programa Abulafia.

Abulafia era un programa que aparecía en la novela "El péndulo de Foucault" de Umberto Eco. El programa se dedicaba a combinar aleatoriamente palabras para obtener frases con sentido más o menos bien conseguido. Puede usarse para obtener argumentos originales para posteriormente desarrollar una historia mejor hilvanada, o simplemente como diversión.

El programa debe contar con varias categorías de palabras (al menos estas: artículos, sustantivos, adjetivos, verbos y conjunciones). Combinándolas al azar, podrá obtener frases cortas y sintácticamente correctas, aunque tengan significados absurdos en apariencia y los verbos no estén bien conjugados (se pueden dejar en infinitivo, como hablan los indios en las películas del oeste)
